  # STUDENT WALLET SYSTEM


### Project Description
A Python-based Student Wallet System that allows users to securely manage a digital wallet with features such as signup, login, deposit, withdrawal, money transfer, and transaction tracking. The system is built using Python and MySQL, and demonstrates database connectivity, transaction handling, and modular programming.
